# Email Samples

Apple製品の注文〜配達までのメールサンプル集。  
Cloudflare Email Workerでのパース処理開発用。

## フォルダ構成

| フォルダ | 説明 | 連絡先メール | 購入元 |
|---------|------|-------------|--------|
| apple-account | Apple ID紐付き注文 | iCloudメール | Apple Store |
| apple-guest-gmail | Appleゲスト注文 | Gmail | Apple Store |
| amazon | Amazon経由購入 | Gmail | Amazon |

## メールの流れ

### Apple Store（apple-account / apple-guest-gmail）

| # | メール種類 | 件名パターン | トリガー |
|---|-----------|-------------|----------|
| 01 | 注文確認 | `ご注文の確認 - Wxxxxxxxxxx` | 注文直後 |
| 02 | 注文完了 | `ご注文ありがとうございます Wxxxxxxxxxx` | 決済確定後 |
| 02a | お届け日変更 | `Appleよりご注文に関するお知らせ：Wxxxxxxxxxx` | お届け日変更時 |
| 03 | 出荷通知 | `お客様の商品は配送中です。ご注文番号: Wxxxxxxxxxx` | 出荷時 |
| 04 | 請求書 | `請求金額のお知らせ: AERxxxxxxx` | 出荷後（PDF添付） |
| 05 | 配達完了 | ※未取得 | 配達完了時 |

### Amazon

| # | メール種類 | 件名パターン | トリガー |
|---|-----------|-------------|----------|
| 01 | 注文済み | `注文済み: [商品名]...` | 注文直後 |
| 02 | 発送済み | `発送済み: [商品名]...` | 発送時 |
| 03 | 配達中 | `配達中: [商品名]...` | 配達中 |
| 04 | 配達済み | `配達済み 1点の商品 - 注文番号 xxx-xxxxxxx-xxxxxxx` | 配達完了時 |

## ファイル命名規則

### Apple Store
```
{連番}_{メール種類}_{注文番号}.pdf
{連番}_{メール種類}_{注文番号}_{請求番号}_{email|attachment}.pdf

例:
01_order_confirmation_W1399071475.pdf
04a_invoice_pdf_W1399071475_MC42724146.pdf
```

### Amazon
```
{連番}_{メール種類}_{注文番号}.eml

例:
01_order_250-8477857-2415055.eml
04_delivered_250-8477857-2415055.eml
```

## 注意事項

- 個人情報（住所、電話番号、メールアドレス）はマスク済み
- シリアル番号、IMEI等はサンプルのまま（実機と紐付かない）
